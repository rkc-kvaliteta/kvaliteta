Document

Using the CiD Framework, we have developed our own Document Authentication System. We developed this using the document libraries which already comprises of more than 4000 types of document patterns and format from all over the world.

It will run the following tests on the documents to recognize the type of the document they are:-

  • VIZ Reading
  • MRZ Reading
  • Thread Detect
  • Microprint Test
  • Colour Check
  • B900 Ink Check
  • Security Paper Check
  • Optically Variable Inks (OVI)
  • Optically Variable Devices
  • Document Classification

Number Plate

Automatic number-plate recognition (ANPR) is a traffic surveillance method based on optical character recognition (OCR). A specific OCR algorithm processes captured images or footage to recognize the plate characters.

Automatic number-plate recognition (ANPR) can be implemented in any traffic related application using an existing CCTV/IP camera system.

Advanced traffic monitoring cameras can detect the following traffic offenses:

  • Solid lane crossing
  • Wrong-way driving
  • Red light crossing
  • Not using safety belts
  • Exceeding speed limit
